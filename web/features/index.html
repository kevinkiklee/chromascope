Features

Everything inside Chromascope

Professional-grade color analysis for creative workflows.

Analysis

Color Spaces

YCbCr (BT.601)

Broadcast-standard color space used in video production. Separates luma (Y) from chroma (Cb, Cr) for precise saturation analysis.

CIE LUV

Perceptually uniform color space where equal distances represent equal perceived color differences. Ideal for colorimetric work.

HSL

Hue-Saturation-Lightness — an intuitive representation of color suitable for artistic grading workflows.

Visualization

Density Modes

Scatter

Each pixel plotted as a point on the vectorscope. Provides an unweighted view of the full color gamut in the image.

Heatmap

Pixel density is accumulated and color-mapped — areas with more pixels glow brighter, revealing dominant hues at a glance.

Bloom

Glow effect applied to high-density clusters, making tightly grouped color populations visually distinctive.

Overlays

Graticule

Configurable circular grid with saturation rings at standard intervals, hue labels at 30° increments, and a fine angular grid.

Reference

Skin Tone Line

Industry-standard reference line for accurate skin tone reproduction. Essential for portrait retouching and broadcast-compliant color work.

Interactive

Color Harmony Zones

Overlay complementary, split-complementary, triadic, tetradic, and analogous harmony zones. Rotate zones interactively and use fit-to-scheme to push colors toward your chosen harmony.

Platform Support

Adobe Photoshop

UXP panel plugin. Samples the active document on each pixel-level edit event. Supports all document color modes.

Lightroom Classic

LrC SDK panel plugin. Responds to photo selection and develop module slider changes in real time.
